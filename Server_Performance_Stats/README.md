Скрипт для анализа базовой производительности сервера.

## Описание
Этот скрипт анализирует:
- Использование CPU
- Использование памяти (RAM)
- Использование дискового пространства
- Топ-5 процессов по использованию CPU
- Топ-5 процессов по использованию памяти

## Использование
```bash
chmod +x server-stats.sh
./server-stats.sh
